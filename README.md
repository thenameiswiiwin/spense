## Getting Started

### Clone Repo

```
git clone https://github.com/thenameiswiiwin/spense.git
```

### Change Directory

```
cd spense
```

### Install Dependencies

```
npm install
```

### Scripts

##### Run StyleLint

```
npm run lint:css
```

##### Run ESLint

```
npm run lint:js
```

##### Run ESLint & StyleLint

```
npm run lint
```

##### Prettier

```
npm run format
```
